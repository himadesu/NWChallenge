Instructions to execute the tests:

1) npm install
installs the required node modules 
2) npm install -g webdriver-manager
installs the required web driver manager
3) webdriver-manager update --standalone 
downloads required Selenium for running Java Scripts
4) webdriver-manager start
To Start Selenium Server 
5)npm run start-api
Runs the Java Script for Web Service Challenge
6)npm run start-ui
Runs the Java Script for UI Challenge
